# HEY-MAP
내가 살기 좋은 동네를 점수로 알려준다
##
GIS 기반으로 생활권에 있는 지도에 매핑된 정보를 통합해서 분석한다면 동네에 점수를 매길 수 있겠다는 아이디어로 시작했다. 공공데이터 활용 공모전을 준비하며 시작하게 되었고 파이썬 기반 기술스텍을 쌓았다.
